🧩 Installation des Scripts

Pour profiter pleinement du Multivision sans publicités et avec le lecteur vidéo isolé, vous devez configurer votre navigateur correctement. Suivez ces étapes :

🦊 Recommandation : Firefox + uBlock Origin

Nous recommandons fortement d'utiliser Firefox. Bien que l'application fonctionne sur Chrome ou Edge, le bloqueur de publicités (uBlock Origin) y est beaucoup moins efficace en raison des récentes restrictions de Google (Manifest V3).

Étape 1 : Installer le bloqueur de publicités (uBlock Origin)

Ceci est essentiel pour bloquer les publicités intrusives des sites de streaming.

Firefox (uBlock) Chrome (uBlock) Edge (uBlock)
Étape 2 : Installer le gestionnaire de scripts (Tampermonkey)

Cette extension permet de faire fonctionner notre script de nettoyage vidéo.

Firefox (Tampermonkey) Chrome (Tampermonkey) Edge (Tampermonkey)
Étape 3 : Installer le Script de nettoyage

Une fois Tampermonkey installé, cliquez sur le bouton ci-dessous. Tampermonkey s'ouvrira, cliquez alors sur "Installer" ou "Mettre à jour".
